# Finance Tracker
A Finance Tracker web application built using Flask and SQLite.

## Features

-User Registration
-User Login
-Add Income
-Edit Income
-Delete Income
-Edit Expense
-Delete Expense
-Budget managemnet
-Reports
-Pie Chart
-PDF Export
-Excel Export

## Technologies Used

-Python
-Flask
-SqLite
-HTML
-CSS
-JavaScript
-chart.js
